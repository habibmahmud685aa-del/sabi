SABI MVP (Clever AI CBT Engine)
==========================
Это тестовая версия умного веб-приложения чат-бота для психологической самопомощи, основанного на принципах когнитивно-поведенческой терапии (КПТ) и генеративного ИИ.

🔥 НОВЫЕ ФУНКЦИИ (Этапы 1-4):
* База данных SQLite для сохранения истории чата и "Динамики" настроения.
* Умный движок контекста (память на последние 6 сообщений).
* Аналитика состояния со встроенными графиками.
* Поддержка OpenAI API для умной генерации эмпатичных ответов.

--------------------------
ШАГ 1: ПОДГОТОВКА (Один раз)
--------------------------
1) Открой терминал внутри папки с проектом `sabi-mvp-clean`
2) Создай и активируй виртуальное окружение:
   (Windows)
   > python -m venv .venv
   > .venv\Scripts\activate
   
   (Mac/Linux)
   > python3 -m venv .venv
   > source .venv/bin/activate

3) Установи все нужные библиотеки (Flask, OpenAI, dotenv):
   > pip install -r requirements.txt

--------------------------
ШАГ 2: НАСТРОЙКА AI (Обязательно)
--------------------------
Чтобы Саби стала по-настоящему "умной", ей нужен ключ от нейросети.
1) В папке проекта найди файл `.env.example`.
2) Переименуй его просто в `.env`.
3) Открой `.env` в блокноте и замени текст на свой реальный API-ключ от OpenAI:
   OPENAI_API_KEY=sk-твой_длинный_секретный_ключ

*Примечание: Если ты не укажешь ключ или оставишь .env.example, сервер всё равно запустится, но Саби будет отвечать строгими, заранее прописанными шаблонами.*

--------------------------
ШАГ 3: ЗАПУСК СЕРВЕРА
--------------------------
1) Убедись, что виртуальное окружение активировано (в терминале слева должно быть написано (.venv) зеленым цветом).
2) Запусти бэкенд проекта:
   > python app.py
3) В консоли появится надпись `Running on http://127.0.0.1:8000`. 
4) Открой браузер (Chrome/Edge/Safari) и перейди по ссылке:
   http://127.0.0.1:8000

База данных `data/sabi.db` создастся автоматически при первом запуске!
